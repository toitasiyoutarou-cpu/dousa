YT Serve Click Analyzer v5

目的:
YouTube上のバレーボール・サーブ動画を表示し、その上に重ねた透明キャンバスをクリックして、ボール軌道、中心線からの左右ズレ、打点、トス、落下点などを観察分析するためのアプリです。

重要:
- YouTube動画の映像データを直接取得・自動解析するものではありません。
- 表示されたYouTube動画の上からクリックし、クリック座標と動画時刻を記録します。
- YouTube IFrame APIを使うため、インターネット接続が必要です。
- クリック解析ONの間は透明キャンバスが上に重なるため、YouTube本体の操作はしにくくなります。アプリ内の再生/停止・フレーム送りボタンを使ってください。

使い方:
1. index.html をブラウザで開く。
2. YouTube URLを貼り付けて「動画を表示」。
3. 中心線2点をクリックして、後方視点の基準線を設定。
4. 必要なら基準線2点をクリックし、実距離mを入力。
5. ボール軌道モードで、コマ送りしながらボール中心をクリック。
6. トス、打点、落下点なども必要に応じてクリック。
7. サマリー、軌道グラフ、左右ズレ推移を見て考察。
8. CSV出力。

探究に向く問い:
- サーブの軌道は中心線からどれくらいずれるか。
- 打点やトス位置と、ボールの左右ズレは関係するか。
- フローターサーブでは、どの時点から軌道がぶれ始めるか。
- 回転が少ないように見えるサーブは、軌道の左右ズレが大きいか。
