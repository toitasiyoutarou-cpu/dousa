操作:
1. 動画を選ぶ
2. AI解析を開始
3. 打点候補を確認

5. 軌道・接触・回転を解析
6. 3Dビューをドラッグして360度回転

注意:
回転数・接触位置・奥行きは30fps単眼動画からのAI推定です。測定値と推定値は画面上で分けて表示します。
