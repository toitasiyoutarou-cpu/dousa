# VolleyMotion Lab v5

30fpsのバレーボールサーブ動画から、身体・手・打点候補を解析し、手首周辺の画像特徴からボール候補を自動検出します。

## 主な機能
- 身体33点・利き手21点
- 打点候補の自動推定
- ボール候補の自動検出（手動修正可能）
- ボール追跡、打ち出し角、左右変化、落下傾向
- 接触前後5コマの自動拡大表示
- 手の進入方向とボール軌道の矢印表示
- 複数サーブ比較、CSV保存

## 注意
自動ボール検出は汎用の物体検出AIではなく、打点付近の円形・輪郭・色差・距離を使った候補推定です。背景や画質によって誤検出するため、緑の候補円を確認し、必要なら手動指定してください。YouTube URLは比較再生専用で、解析には動画ファイルが必要です。
