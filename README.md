# VolleyMotion Lab v2

ジャンプサーブを「身体・手・ボール」の3層で解析する、ブラウザ用プロトタイプです。

## 追加した機能
- 左利き（宮浦健人選手を想定）を初期設定
- MediaPipe Pose Landmarkerによる身体33点
- 打点付近のMediaPipe Hand Landmarkerによる手21点
- 打点候補の自動推定と手動修正
- ボール中心を1回クリックして色特徴で前後フレームを追跡
- ボール軌跡・相対速度
- 手のひら中心とボール中心のずれ
- ボール表面のリング状画像特徴から回転数を試験推定
- 推定回転数の信頼度表示
- CSV・静止画保存
- YouTube参考動画の横並び表示

## 起動
GitHub Pagesに配置するか、ローカルサーバーで開いてください。

Windows: `start_windows.bat`
macOS: `start_mac.command`

またはターミナルでこのフォルダへ移動して:

```bash
python -m http.server 8000
```

ブラウザで `http://localhost:8000` を開きます。

## 操作
1. 動画を読み込む
2. 利き腕と解析範囲を設定
3. 「身体と手を自動解析」
4. 打点候補をスローで確認し、必要なら修正
5. 打点付近で「ボールを指定」を押してボール中心をクリック
6. 「自動追跡・回転推定」

## 撮影条件
- 身体解析: 60fps以上推奨
- ボール回転: 120〜240fps推奨
- 明るい場所、短いシャッター時間、ボール表面模様が見える画角
- 後方動画に加え、利き腕側の斜め側方動画があると手の解析精度が上がります

## 注意
回転数は研究用計測器による値ではなく、単眼動画の表面模様変化から求める試験的な推定です。低fps、モーションブラー、圧縮、ボールの遮蔽によって誤差が大きくなります。必ず信頼度と映像を一緒に確認してください。
