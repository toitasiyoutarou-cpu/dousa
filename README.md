# VolleyMotion Lab MVP

バレーボールのサーブ動画をブラウザ内で解析し、身体の骨格点、関節角度、打点候補、動作局面を表示する試作アプリです。

## できること

- MP4 / MOV / WebM 動画の読み込み
- MediaPipe Pose Landmarkerによる身体33点の自動検出
- 利き腕の肘角度、膝角度、肩の傾き、体幹の左右傾きの計算
- 利き腕の高さ・伸展・速度を使った打点候補の自動推定
- 打点候補の手動修正
- 角度グラフ表示
- CSV出力、骨格表示付き静止画出力
- YouTube URLの埋め込み表示（参考・目視比較用）

## 起動方法

### 方法1：GitHub Pages（おすすめ）

1. このフォルダの中身をGitHubリポジトリへアップロードします。
2. GitHubの `Settings` → `Pages` を開きます。
3. `Deploy from a branch` を選び、`main` / `(root)` を公開します。
4. 発行されたURLをChromeまたはEdgeで開きます。

### 方法2：Windowsでローカル起動

1. Pythonが入っているPCで `start_windows.bat` をダブルクリックします。
2. ブラウザで `http://localhost:8000` を開きます。

HTMLファイルの直接ダブルクリックでも動く場合がありますが、ブラウザのセキュリティ設定によりAIモデルが読み込めないことがあるため、HTTPサーバー経由を推奨します。

## 推奨撮影条件

- 後方中央から三脚等で固定
- 選手の全身を常に画面内に入れる
- 可能なら60fps
- 明るい場所で撮影
- 背景と服の色を分ける
- 解析区間はサーブ1回分（3〜8秒程度）に絞る

## 重要な制限

- 現在の「打点候補」はボール接触を直接検出していません。利き腕の手首の高さ、肘の伸展、手首速度から推定します。
- YouTube動画は埋め込み再生のみです。YouTube映像を直接取り出してAI解析する機能はありません。
- 角度は単眼カメラによる推定値です。絶対的な評価より、同じ撮影条件での動画間比較に向いています。
- 初回起動時はMediaPipeのライブラリ、WASM、学習済みモデルをインターネットから読み込みます。
