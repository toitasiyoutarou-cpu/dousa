# VolleyVision AI 2.0

30fpsのバレーボール・サーブ動画から、手の進入角、手のひら角、接触位置、打ち出し角、左右変化、落下傾向を端末内で解析する、GitHub Pages対応のWebアプリです。

## GitHub Pagesへの設置

1. ZIPを展開します。
2. `index.html`、`styles.css`、`app.js`をリポジトリ直下へアップロードします。
3. GitHubの `Settings → Pages` で、`Deploy from a branch`、`main`、`/(root)`を選択します。
4. 公開URLを開き、画面上部が `VolleyVision AI` になっていることを確認します。

## 操作

1. 動画を選択またはドロップ
2. 打点で停止して「この瞬間を打点にする」
3. 画面案内に従い6点をクリック
4. 「軌道を解析する」
5. 結果・軌道・比較表を確認

## 注意

- MP4（H.264）が最も安定します。
- 30fpsでは正確な回転数は出しません。接触条件と軌道変化の比較を目的とします。
- 自動追跡は画像テンプレートによる推定です。背景やブレによって誤差が生じます。
- 動画はブラウザ内で処理し、外部サーバーには送信しません。
